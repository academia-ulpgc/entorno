# Guion temporal V.3 - Copilot con contexto e instrucciones

## Uso

Documento para el docente. No entregar como ficha de estudiante.

## Duracion

1 hora y 15 minutos.

## Objetivo de conduccion

Enseñar que Copilot responde mejor cuando recibe contexto, limites y formato esperado. Reforzar la idea de que la persona dirige y valida.

## Guion temporal

- 0:00-0:10. Activacion:
  - Comparar una peticion vaga con una precisa.
  - Mostrar que "mejora esto" da menos control que una instruccion con contexto.

- 0:10-0:30. Redaccion:
  - Revisar un texto con Copilot.
  - Pedir cambios limitados.
  - Hacer que el estudiante anote cambios aceptados y rechazados.

- 0:30-0:45. Documentacion:
  - Generar esquema o resumen a partir de hechos.
  - Reforzar: no inventar pasos no realizados.
  - Comprobar si la respuesta respeta el formato pedido.

- 0:45-1:05. Contexto:
  - Crear `contexto-del-proyecto.txt`.
  - Crear `instrucciones-copilot.txt`.
  - Probar una respuesta con contexto.
  - Explicar que contexto no es garantia absoluta, pero mejora la direccion.

- 1:05-1:15. Validacion:
  - Comparar respuestas.
  - Escribir reglas personales de uso responsable.
  - Anticipar V.4: usar IA y Python para generar un informe desde datos ficticios.

## Mensaje clave

La competencia no es solo usar Copilot, sino dirigirlo: contexto, objetivo, limites, formato y revision humana.
