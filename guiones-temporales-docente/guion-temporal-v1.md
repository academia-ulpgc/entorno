# Guion temporal V.1 - VS Code en español, LaTeX y Copilot

## Uso

Documento para el docente. No entregar como ficha de estudiante.

## Duracion

1 hora y 15 minutos.

## Objetivo de conduccion

Introducir Visual Studio Code como herramienta de trabajo de las proximas sesiones y usar LaTeX para explicar tres ideas base: carpeta de proyecto, codigo fuente y compilacion.

## Guion temporal

- 0:00-0:10. Presentacion:
  - Recordar que el curso ya ha empezado y que esta fase es mas practica.
  - Presentar VS Code como mesa de trabajo.
  - Explicar la idea: carpeta, codigo fuente, compilacion y resultado.
  - Conectar con la idea de fondo: la IA como apoyo tecnico para automatizar.

- 0:10-0:22. Comprobacion inicial:
  - Comprobar que VS Code abre correctamente.
  - Comprobar si LaTeX esta disponible con `pdflatex --version`.
  - Comprobar acceso a cuenta institucional, SSO o cuenta autorizada.
  - Anotar incidencias sin detener toda la sesion.

- 0:22-0:37. Idioma y extensiones:
  - Instalar Spanish Language Pack.
  - Instalar LaTeX Workshop.
  - Instalar o activar Copilot.
  - Reforzar que las extensiones añaden capacidades al entorno.

- 0:37-0:55. Documento LaTeX:
  - Crear `curso-vscode`.
  - Crear `informe_prueba.tex`.
  - Explicar que `.tex` es codigo fuente.
  - Presentar el ejemplo como un informe municipal ficticio de seguimiento de incidencias.
  - Compilar y abrir el PDF.
  - Señalar que el PDF es el resultado compilado, no el archivo de trabajo.

- 0:55-1:10. Copilot:
  - Iniciar sesion con SSO o cuenta autorizada.
  - Pedir a Copilot explicacion del documento LaTeX.
  - Pedir que explique diferencia entre `.tex` y PDF.
  - Si hay error de compilacion, usarlo como ejemplo de diagnostico asistido.

- 1:10-1:15. Cierre:
  - Revisar checklist final.
  - Recoger incidencias.
  - Anticipar la sesion V.2: misma logica de proyecto, documentos y resultados revisables.

## Mensaje clave

LaTeX permite entender una idea esencial para automatizar: escribir instrucciones en un archivo, procesarlas y obtener un resultado revisable.
