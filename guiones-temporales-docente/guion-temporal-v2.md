# Guion temporal V.2 - Primer proyecto con LaTeX y Python

## Uso

Documento para el docente. No entregar como ficha de estudiante.

## Duracion

1 hora y 15 minutos.

## Objetivo de conduccion

Consolidar la idea de proyecto como carpeta organizada y conectar la compilacion de LaTeX con la ejecucion de Python.

## Guion temporal

- 0:00-0:10. Repaso:
  - Recuperar la secuencia de V.1: carpeta, codigo, compilacion, resultado.
  - Comprobar rapidamente VS Code, LaTeX, Python y Copilot.
  - Explicar que ahora habra varios archivos en el mismo proyecto.

- 0:10-0:25. Proyecto:
  - Crear carpeta `practica-v2`.
  - Crear `descripcion.txt`, `informe.tex`, `analisis.py` y `resultados.txt`.
  - Explicar para que sirve cada archivo.

- 0:25-0:40. Texto plano:
  - Crear `descripcion.txt`.
  - Explicar que tambien se documenta el trabajo en texto sencillo.
  - Evitar formatos complejos: usar listas simples.

- 0:40-1:00. LaTeX:
  - Crear `informe.tex`.
  - Compilar y revisar PDF.
  - Reforzar la diferencia entre archivo fuente y resultado.

- 1:00-1:10. Python:
  - Crear `analisis.py`.
  - Ejecutar el script.
  - Copiar salida en `resultados.txt`.
  - Comparar: en LaTeX se compila; en Python se ejecuta.

- 1:10-1:15. Cierre:
  - Revisar checklist.
  - Dejar carpeta ordenada.
  - Anticipar V.3: usar Copilot con mas contexto y mejores instrucciones.

## Mensaje clave

Un proyecto de automatizacion no es un archivo suelto: es una carpeta con entradas, instrucciones, resultados y comprobaciones.
